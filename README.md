# 7 Days to Die Mod Profile Launcher

**Never manually copy mods again!** Switch between Vanilla, Darkness Falls, and custom mod sets with one click.

![.NET 8](https://img.shields.io/badge/.NET-8.0-512BD4)
![Windows](https://img.shields.io/badge/Platform-Windows%2010%2F11-0078D6)

## What This Does

This launcher lets you create "profiles" for different mod configurations and switch between them instantly. No more:
- Copying mods in and out manually
- Maintaining duplicate 10GB+ game installations
- Forgetting which mods you had installed

Instead:
- One-click profile switching
- Automatic mod injection/removal
- Keep all your mod sets organized
- Launch directly into the right configuration

## Features

- **Profile Management** - Create unlimited mod configurations
- **One-Click Launch** - Select profile and go
- **EAC Toggle** - Enable/disable Easy Anti-Cheat per profile
- **Drag & Drop Mods** - Add mods by dragging folders or ZIP files
- **Mod Browser** - View mod details (name, author, version) from ModInfo.xml
- **Auto-Detection** - Finds your 7DTD installation automatically
- **Profile Export/Import** - Share profiles with friends
- **System Tray** - Minimizes to tray while game is running
- **Protected Mods** - Core mods like 0_TFP_Harmony are never touched

## Quick Start

1. **Download** the latest release from [Releases](../../releases)
2. **Run** `7DTDModLauncher.exe`
3. **Set** your 7 Days to Die installation path (auto-detected if using Steam)
4. **Create** your first profile
5. **Add mods** via drag & drop or the Manage Mods button
6. **Launch** and play!

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Enter` | Launch game |
| `Ctrl+N` | New profile |
| `Ctrl+M` | Manage mods |
| `Ctrl+,` | Settings |
| `F5` | Refresh profiles |
| `Delete` | Delete profile |
| `F1` | About |

## System Requirements

- Windows 10/11 (x64)
- 7 Days to Die installed (Steam)
- ~500MB free space per profile
